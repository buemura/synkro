# @synkro/core v0.14.0

**Release date:** 2026-03-07

## Changes

### IMP-04: Event schema validation `[SEC]`

Added schema validation for event payloads at both publish and handler dispatch time. `SynkroOptions` now accepts a `schemas` map (`Record<string, SchemaValidator>`) for global event-type validation. Per-event schemas can also be set via the `schema` field on `SynkroEvent`. `SchemaValidator` is a simple `(payload: unknown) => void` function that throws on invalid input, compatible with any validation library (Zod, Joi, etc.).

- **Publish time:** throws synchronously if the payload fails validation
- **Handler dispatch (global schema):** drops invalid messages without retry
- **Handler dispatch (per-entry schema):** triggers failure event (subject to retry policy)

### FT-06: Workflow timeout

`SynkroWorkflowStep` and `SynkroWorkflow` now support an optional `timeoutMs` field. When a step exceeds its timeout, a synthetic failure event is published, triggering the `onFailure` branch or marking the workflow as failed. Step-level timeout overrides workflow-level timeout.

### IMP-06: Graceful shutdown

`stop()` now drains active handlers before disconnecting the transport. It polls handler and workflow registry active counts every 50ms until all complete or the configurable `drainTimeout` (default 5000ms) is reached. A warning is logged if forced to disconnect with active handlers remaining. Configure via `SynkroOptions.drainTimeout`.

### TD-06: Instance-scoped logger

Logger is now a class (`Logger`) with instance-level `debugEnabled` state. Each `Synkro` instance creates its own `Logger` and passes it to all registries and transport. Multiple instances with different `debug` settings no longer conflict. The `Logger` class is exported for external use.

### TD-03: In-memory transport TTL (verified)

Confirmed that `InMemoryManager` correctly applies TTL via `applyTtl()` in `setCache()` and `setCacheIfNotExists()`, with lazy eviction on read. The original unused `_ttlSeconds` parameter was already fixed in a prior change.
