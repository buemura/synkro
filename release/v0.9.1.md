# @synkro/core v0.9.1

**Release date:** 2026-03-07

## Changes

### TD-11: Validate workflow definitions at registration

`registerWorkflows` now validates each workflow before registration and throws clear errors for:

- Empty workflow name
- Empty `steps` array
- Duplicate step types within a workflow
- Dangling `onSuccess`/`onFailure` targets that reference non-existent steps

This prevents invalid workflows from being silently registered and catching structural issues at startup rather than at runtime.
