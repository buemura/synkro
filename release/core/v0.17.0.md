# @synkro/core v0.17.0

**Release date:** 2026-03-10

## Changes

### IMP-09: Transport contract test suite

Shared test suite (`transportContractTests`) that validates `InMemoryManager` against the full `TransportManager` interface. Covers pub/sub round-trips, cache CRUD with TTL expiry, `setCacheIfNotExists` atomicity, increment, list operations (including negative indices), `deleteKey`, and `disconnect` cleanup. 24 contract tests ensure transport implementations behave identically.

### FT-12: Event versioning

First-class support for versioned event types. Events following the `base:event:vN` convention (e.g., `user:created:v2`) now trigger automatic base-event fanout:

- Publishing `user:created:v2` delivers to both the `user:created:v2` channel and the `user:created` base channel.
- Handlers registered on the base event (`user:created`) receive all versions as a catch-all.
- Handlers on a specific version (`user:created:v2`) receive only that version.
- Unversioned events continue to work exactly as before (non-breaking).

New public exports: `parseEventType()`, `isVersionedEvent()`, and `ParsedEventType`.
